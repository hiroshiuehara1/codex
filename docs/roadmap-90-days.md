- issue/PR templates

- branch protection automation baseline
- rollback guardrails in release/incident agents

## Weeks 5-6

- tester agent test-plan generation
- CI check matrix + security scanning

## Weeks 7-8

- reviewer policy gate workflow
- merge gate logic for risk-aware approvals

## Weeks 9-10

- release agent workflow with staged progression
- canary health threshold enforcement

## Weeks 11-12

- incident monitor workflow
- governance reporting workflow
- artifacts for go/no-go pilot review
